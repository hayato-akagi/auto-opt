# bolt-service

ボルト締結前の初期位置からレンズ位置ずれを計算するサービス。位置依存の非線形応答とノイズでリアルな締結挙動を再現。

- **Port**: 9005（コンテナ内は8005）
- **技術スタック**: Python, FastAPI, NumPy
- **依存サービス**: position-service（初期位置 x0, y0 を受け取る）
- **モデルバージョン**: v3.0（位置依存べき乗モデル）

> **v2.0 からの変更点**:
> - トルク依存 → 位置依存モデルに変更
> - 入力: `torque_upper`, `torque_lower` → `x0`, `y0`
> - パラメータ: `a`, `b`, `c`, `d` → `a_x`, `b_x`, `a_y`, `b_y`（XY独立）
> - ノイズ: σ(T) → σ(|x0|)（絶対値で対称性確保）

## 概要

ボルト締結時の位置ずれを、初期位置依存のべき乗関数とノイズでモデル化：

```
Δx = a_x × x0^b_x + N(0, σ_x(|x0|)²)
Δy = a_y × y0^b_y + N(0, σ_y(|y0|)²)

σ(|x0|) = max(0, σ_base + σ_prop × |x0|)
```

- **非線形応答**: べき指数 b_x, b_y で飽和/加速挙動を表現
- **位置依存ノイズ**: 大きな初期位置でばらつきも増加
- **対称性**: 絶対値 |x0| でノイズが対称
- **方向性制御**: 係数 a_x, a_y の符号で変位方向を設定

## API

### `POST /bolt/apply`

#### Request

```jsonc
{
  "x0": 0.02,                // mm, ボルト締結前のX位置
  "y0": -0.05,               // mm, ボルト締結前のY位置
  "bolt_model": {
    "upper": {
      // X方向べき乗モデル: Δx = a_x × x0^b_x
      "a_x": 0.05,             // X方向係数 (無次元), 範囲: -0.5~0.5
      "b_x": 1.0,              // X方向べき指数, 範囲: 0<b_x≤2
      
      // Y方向べき乗モデル: Δy = a_y × y0^b_y
      "a_y": 0.08,             // Y方向係数 (無次元), 範囲: -0.5~0.5
      "b_y": 1.0,              // Y方向べき指数, 範囲: 0<b_y≤2
      
      // 位置依存ノイズ: σ(|x0|) = σ_base + σ_prop × |x0|
      "noise_base_x": 0.002,   // X軸ベースノイズ (mm), ≥0
      "noise_prop_x": 0.001,   // X軸位置比例ノイズ (無次元), ≥0
      "noise_base_y": 0.005,   // Y軸ベースノイズ (mm), ≥0
      "noise_prop_y": 0.002    // Y軸位置比例ノイズ (無次元), ≥0
    },
    "lower": {
      "a_x": -0.03,
      "b_x": 1.0,
      "a_y": 0.05,
      "b_y": 1.0,
      "noise_base_x": 0.001,
      "noise_prop_x": 0.0005,
      "noise_base_y": 0.003,
      "noise_prop_y": 0.001
    }
  },
  "random_seed": null          // 再現性が必要な場合に指定
}
```

#### Response (200)

```jsonc
{
  "delta_x": 0.003,            // mm, X方向ずれ合計
  "delta_y": 0.008,            // mm, Y方向ずれ合計
  "used_seed": 1234567890,     // 実際に使用されたシード（再現用）
  "detail": {
    "upper": {"delta_x": 0.0015, "delta_y": 0.0055},
    "lower": {"delta_x": 0.0015, "delta_y": 0.0025}
  }
}
```

## 計算モデル（v3.0）

### 基本式

上下ボルトそれぞれについて:

```
Δx_i = a_x,i × x0^b_x,i + N(0, σ_xi(|x0|)²)
Δy_i = a_y,i × y0^b_y,i + N(0, σ_yi(|y0|)²)
```

位置依存ノイズ:
```
σ_x(|x0|) = max(0, σ_base_x + σ_prop_x × |x0|)
σ_y(|y0|) = max(0, σ_base_y + σ_prop_y × |y0|)
```

合計:
```
Δx = Δx_upper + Δx_lower
Δy = Δy_upper + Δy_lower
```

### パラメータの意味

| パラメータ | 役割 | 典型値 |
|-----------|------|--------|
| `a_x`, `a_y` | 変位の方向と大きさ | ±0.03 ~ ±0.1 |
| `b_x`, `b_y` | 応答特性（1=線形、<1=飽和、>1=加速） | 0.7 ~ 1.5 |
| `noise_base` | 位置ゼロ時のばらつき | 0.001 ~ 0.005 mm |
| `noise_prop` | 位置依存ばらつき | 0 ~ 0.003 |

### モデルの挙動例

**線形応答（b_x=1）:**
```
x0 = 0.1 mm → Δx = 0.05 × 0.1^1.0 = 0.005 mm
x0 = 0.5 mm → Δx = 0.05 × 0.5^1.0 = 0.025 mm （5倍）
```

**飽和的応答（b_x=0.7）:**
```
x0 = 0.1 mm → Δx = 0.05 × 0.1^0.7 ≈ 0.010 mm
x0 = 0.5 mm → Δx = 0.05 × 0.5^0.7 ≈ 0.034 mm （3.4倍、線形より小）
```

**加速的応答（b_x=1.5）:**
```
x0 = 0.1 mm → Δx = 0.05 × 0.1^1.5 ≈ 0.0016 mm
x0 = 0.5 mm → Δx = 0.05 × 0.5^1.5 ≈ 0.018 mm （11倍、線形より大）
```

**位置依存ノイズ:**
```
σ(|x0|) = max(0, 0.002 + 0.001 × |x0|)

x0 =  0.0 mm → σ = 0.002 mm
x0 = ±0.2 mm → σ = 0.0022 mm
x0 = ±0.8 mm → σ = 0.0028 mm （大きな位置でばらつき増大）
```

**対称性**:
- `x0 = +0.5 mm` と `x0 = -0.5 mm` で同じノイズレベル（|x0| = 0.5）
- 物理的に妥当な挙動

## ノイズ適用タイミング

ノイズは **API 呼び出しごとに毎回新たにサンプリング** される。

- 1回のステップ実行 = 1回の Bolt Service 呼び出し = 1回のノイズサンプリング
- 試行開始時にシードを固定するような仕組みはない
- 再現性が必要な場合は `random_seed` を明示的に指定する

## ノイズシードの挙動

| `random_seed` 値 | 挙動 |
|------------------|------|
| `null` (デフォルト) | 呼び出しごとにランダムなシードを生成。毎回異なる結果 |
| 整数値 (例: `42`) | 指定シードでノイズを生成。同じ入力なら同じ結果 |

レスポンスの `used_seed` に実際に使われたシードが返る。Recipe Service はこの値を step JSON に記録する。

### 再現実験の手順

1. 過去の step JSON から `bolt_shift.used_seed` を取得
2. 同じ初期位置（x0, y0）・ボルトモデル・`random_seed` で再呼出し
3. 同一の `delta_x`, `delta_y` が得られる

## パラメータ範囲

| パラメータ | 単位 | 推奨範囲 | 制約 |
|-----------|------|---------|------|
| `x0`, `y0` | mm | -1.0 ~ 1.0 | 任意 |
| `a_x`, `a_y` | 無次元 | -0.2 ~ 0.2 | -0.5 ~ 0.5 |
| `b_x`, `b_y` | 無次元 | 0.7 ~ 1.5 | 0 < b ≤ 2 |
| `noise_base_x`, `noise_base_y` | mm | 0.001 ~ 0.005 | ≥ 0 |
| `noise_prop_x`, `noise_prop_y` | 無次元 | 0 ~ 0.003 | ≥ 0 |

## 旧モデルからの移行（v2.0 → v3.0）

### 概念の変更

v2.0（トルク依存）からv3.0（位置依存）への主な変更：

- **入力**: トルク `(T_upper, T_lower)` → 初期位置 `(x0, y0)`
- **パラメータ**: 4個 `(a, b, c, d)` → 4個 `(a_x, b_x, a_y, b_y)` に名称変更
- **ノイズ**: σ(T) → σ(|x0|) で対称性確保

**旧パラメータ（v1.0）:**
```json
{
  "shift_x_per_nm": 0.001,
  "shift_y_per_nm": 0.003,
  "noise_std_x": 0.002,
  "noise_std_y": 0.005
}
```

**新パラメータ（v2.0、等価設定）:**
```json
{
  "a": 0.001,              // = shift_x_per_nm
  "b": 1.0,                // 線形
  "c": 0.003,              // = shift_y_per_nm
  "d": 1.0,                // 線形
  "noise_base_x": 0.002,   // = noise_std_x
  "noise_prop_x": 0.0,     // トルク依存なし
  "noise_base_y": 0.005,   // = noise_std_y
  "noise_prop_y": 0.0      // トルク依存なし
}
```

### 推奨設定例

**標準締結（線形 + トルク依存ノイズ）:**
```json
{
  "upper": {
    "a": 0.001, "b": 1.0, "c": 0.003, "d": 1.0,
    "noise_base_x": 0.002, "noise_prop_x": 0.001,
    "noise_base_y": 0.005, "noise_prop_y": 0.002
  },
  "lower": {
    "a": -0.0005, "b": 1.0, "c": 0.002, "d": 1.0,
    "noise_base_x": 0.001, "noise_prop_x": 0.0005,
    "noise_base_y": 0.003, "noise_prop_y": 0.001
  }
}
```

**弾性変形主体（飽和的）:**
```json
{
  "a": 0.002, "b": 0.7,    // 低トルクで敏感、高トルクで飽和
  "c": 0.005, "d": 0.8,
  "noise_base_x": 0.001, "noise_prop_x": 0.0015,
  "noise_base_y": 0.002, "noise_prop_y": 0.003
}
```

**塑性変形含む（加速的）:**
```json
{
  "a": 0.0005, "b": 1.5,   // 高トルクで急増
  "c": 0.001, "d": 1.3,
  "noise_base_x": 0.001, "noise_prop_x": 0.002,
  "noise_base_y": 0.002, "noise_prop_y": 0.004
}
```
