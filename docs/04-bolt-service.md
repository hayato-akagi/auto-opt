# Bolt Service 仕様

- **Port**: 9005（コンテナ内は8005）
- **役割**: ボルト締結前の初期位置からレンズ位置ずれを計算。位置依存の非線形応答とノイズでリアルな変形を再現。
- **依存**: position-service（初期位置 x0, y0 を受け取る）
- **モデルバージョン**: v3.0（位置依存べき乗モデル）

> **v2.0 からの変更点**:
> - トルク依存 → 位置依存モデルに変更
> - 入力: `torque_upper`, `torque_lower` → `x0`, `y0`
> - パラメータ: `a`, `b`, `c`, `d` → `a_x`, `b_x`, `a_y`, `b_y`（XY独立）
> - ノイズ: σ(T) → σ(|x0|)（絶対値で対称性確保）

## API

### `POST /bolt/apply`

#### Request Body

```jsonc
{
  "x0": 0.02,                // mm, ボルト締結前のX位置
  "y0": -0.05,               // mm, ボルト締結前のY位置
  "bolt_model": {
    "upper": {
      // X方向べき乗モデル (Δx = a_x × x0^b_x)
      "a_x": 0.05,             // X方向係数 (無次元)
      "b_x": 1.0,              // X方向べき指数 (無次元)
      
      // Y方向べき乗モデル (Δy = a_y × y0^b_y)
      "a_y": 0.08,             // Y方向係数 (無次元)
      "b_y": 1.0,              // Y方向べき指数 (無次元)
      
      // 位置依存ノイズ (σ(|x0|) = σ_base + σ_prop × |x0|)
      "noise_base_x": 0.002,   // X軸ベースノイズ (mm)
      "noise_prop_x": 0.001,   // X軸位置比例ノイズ (無次元)
      "noise_base_y": 0.005,   // Y軸ベースノイズ (mm)
      "noise_prop_y": 0.002    // Y軸位置比例ノイズ (無次元)
    },
    "lower": {
      "a_x": -0.03,
      "b_x": 1.0,
      "a_y": 0.05,
      "b_y": 1.0,
      "noise_base_x": 0.001,
      "noise_prop_x": 0.0005,
      "noise_base_y": 0.003,
      "noise_prop_y": 0.001
    }
  },
  "random_seed": null          // 再現性が必要な場合に指定
}
```

#### Response (200)

```jsonc
{
  "delta_x": 0.003,            // mm, X方向ずれ合計
  "delta_y": 0.008,            // mm, Y方向ずれ合計
  "used_seed": 1234567890,     // 実際に使用されたシード（再現用）
  "detail": {
    "upper": {"delta_x": 0.0015, "delta_y": 0.0055},
    "lower": {"delta_x": 0.0015, "delta_y": 0.0025}
  }
}
```

#### Error Response (422)

```jsonc
{
  "detail": [
    {"loc": ["body", "x0"], "msg": "field required", "type": "value_error.missing"}
  ]
}
```

### `GET /health`

## 計算モデル（v3.0: 位置依存べき乗モデル）

### 基本式

上下ボルトそれぞれについて:

$$\Delta x_i = a_{x,i} \cdot x_0^{b_{x,i}} + \mathcal{N}(0,\, \sigma_{xi}(|x_0|)^2)$$
$$\Delta y_i = a_{y,i} \cdot y_0^{b_{y,i}} + \mathcal{N}(0,\, \sigma_{yi}(|y_0|)^2)$$

位置依存ノイズの標準偏差（絶対値で対称性を保つ）:

$$\sigma_{xi}(|x_0|) = \max(0,\, \sigma_{\text{base},xi} + \sigma_{\text{prop},xi} \cdot |x_0|)$$
$$\sigma_{yi}(|y_0|) = \max(0,\, \sigma_{\text{base},yi} + \sigma_{\text{prop},yi} \cdot |y_0|)$$

合計:

$$\Delta x = \Delta x_{\text{upper}} + \Delta x_{\text{lower}}$$
$$\Delta y = \Delta y_{\text{upper}} + \Delta y_{\text{lower}}$$

### パラメータ説明

| 記号 | パラメータ名 | 範囲 | 物理的意味 |
|-----|------------|------|-----------|
| $x_0, y_0$ | `x0`, `y0` | -1.0 ~ 1.0 mm | ボルト締結前の初期位置 |
| $a_x, a_y$ | `a_x`, `a_y` | -0.5 ~ 0.5 | 方向係数（符号で変位方向、大きさで感度） |
| $b_x, b_y$ | `b_x`, `b_y` | 0 < b ≤ 2 | べき指数（1=線形、<1=飽和的、>1=加速的） |
| $\sigma_{\text{base}}$ | `noise_base` | ≥ 0 | 位置ゼロ時のノイズ量（固有のばらつき） |
| $\sigma_{\text{prop}}$ | `noise_prop` | ≥ 0 | 位置1mmあたりのノイズ増加量 |

### モデルの特徴

**1. 非線形応答（べき指数 b_x, b_y）**

- **b = 1**: 線形応答（変位は初期位置に比例）
  ```
  Δx = a_x × x0
  例: x0 = 0.1 mm → Δx = 0.05 × 0.1 = 0.005 mm
      x0 = 0.5 mm → Δx = 0.05 × 0.5 = 0.025 mm（5倍の位置で5倍の変位）
  ```

- **b < 1** (例: 0.7): 飽和的応答（初期位置が大きいほど変位率が減少）
  ```
  低位置: 初期応答が鋭敏
  高位置: 変位が飽和傾向
  用途: 弾性復元力が効いてくる場合
  ```

- **b > 1** (例: 1.5): 加速的応答（初期位置が大きいほど変位が急増）
  ```
  低位置: 変位小さい
  高位置: 変位が急激に増加
  用途: 応力集中など非線形効果が支配的な場合
  ```

**2. 位置依存ノイズ（絶対値使用で対称性確保）**

初期位置の絶対値に応じてノイズが増加します：

```
σ(|x0|) = max(0, σ_base + σ_prop × |x0|)

例： x0 =  0.0 mm → σ = 0.002 mm（ベースのみ）
     x0 = ±0.2 mm → σ = 0.002 + 0.001×0.2 = 0.0022 mm
     x0 = ±0.8 mm → σ = 0.002 + 0.001×0.8 = 0.0028 mm
```

**絶対値を使う理由**:
- ボルト締結の物理的挙動は位置の符号で対称（+0.5 mm と -0.5 mm で同じばらつき）
- |x0| によりノイズが対称になる
- max(0, ...) で負の標準偏差を防止

**3. 方向性（係数 a_x, a_y の符号）**

- a_x > 0: 正の初期X → 正方向へさらに変位
- a_x < 0: 正の初期X → 負方向へ戻る変位（復元的）
- 同様に a_y の符号で Y 方向の挙動を制御

上下ボルトで異なる符号を設定することで、相殺または増幅の挙動を制御可能。

### 設定例

**例1: 線形モデル（シンプル）**
```json
{
  "a_x": 0.05, "b_x": 1.0,  // Δx = 0.05 × x0
  "a_y": 0.08, "b_y": 1.0,  // Δy = 0.08 × y0
  "noise_base_x": 0.002, "noise_prop_x": 0.0,  // 固定ノイズ
  "noise_base_y": 0.005, "noise_prop_y": 0.0
}
```

**例2: 弾性復元主体（飽和的）**
```json
{
  "a_x": 0.08, "b_x": 0.7,   // 大きな初期位置で飽和
  "a_y": 0.10, "b_y": 0.8,
  "noise_base_x": 0.001,
  "noise_prop_x": 0.002,     // 位置依存ノイズあり
  "noise_base_y": 0.002,
  "noise_prop_y": 0.003
}
```

**例3: 非線形変形（加速的）**
```json
{
  "a_x": 0.03, "b_x": 1.5,   // 大きな初期位置で急増
  "a_y": 0.04, "b_y": 1.3,
  "noise_base_x": 0.001,
  "noise_prop_x": 0.003,     // 位置大でノイズ増大
  "noise_base_y": 0.002,
  "noise_prop_y": 0.005
}
```

## ノイズ適用タイミング

ノイズは **API 呼び出しごとに毎回新たにサンプリング** される。

- 1回のステップ実行 = 1回の Bolt Service 呼び出し = 1回のノイズサンプリング
- 試行開始時にシードを固定するような仕組みはない
- 再現性が必要な場合は `random_seed` を明示的に指定する

## ノイズシードの挙動

| `random_seed` 値 | 挙動 |
|------------------|------|
| `null` (デフォルト) | 呼び出しごとにランダムなシードを生成。毎回異なる結果になる |
| 整数値 (例: `42`) | 指定シードでノイズを生成。同じ入力なら同じ結果になる |

レスポンスの `used_seed` に実際に使われたシードが返る。
Recipe Service はこの値を step JSON に記録することで、後から同一結果を再現できる。

### 再現実験の手順

1. 過去の step JSON から `bolt_shift.used_seed` を取得
2. 同じ初期位置（x0, y0）・ボルトモデル・`random_seed` で再呼出し
3. 同一の `delta_x`, `delta_y` が得られる

## パラメータ範囲

| パラメータ | 単位 | 推奨範囲 | 制約 | 説明 |
|-----------|------|---------|------|------|
| `x0`, `y0` | mm | -1.0 ~ 1.0 | 任意 | ボルト締結前の初期位置 |
| `a_x`, `a_y` | 無次元 | -0.2 ~ 0.2 | -0.5 ~ 0.5 | 方向係数（符号で向き） |
| `b_x`, `b_y` | 無次元 | 0.7 ~ 1.5 | 0 < b ≤ 2 | べき指数（1=線形） |
| `noise_base_x`, `noise_base_y` | mm | 0.001 ~ 0.005 | ≥ 0 | ベースノイズ |
| `noise_prop_x`, `noise_prop_y` | 無次元 | 0 ~ 0.003 | ≥ 0 | 位置比例ノイズ |
| `random_seed` | - | - | ≥ 0 or null | 再現用シード |

### 推奨設定値

**標準的な締結（線形近似）:**
```json
{
  "upper": {"a_x": 0.05, "b_x": 1.0, "a_y": 0.08, "b_y": 1.0, 
            "noise_base_x": 0.002, "noise_prop_x": 0.001,
            "noise_base_y": 0.005, "noise_prop_y": 0.002},
  "lower": {"a_x": -0.03, "b_x": 1.0, "a_y": 0.05, "b_y": 1.0,
            "noise_base_x": 0.001, "noise_prop_x": 0.0005,
            "noise_base_y": 0.003, "noise_prop_y": 0.001}
}
```

**不安定な締結（高ノイズ）:**
```json
{
  "noise_base_x": 0.005,    // ベースノイズ大
  "noise_prop_x": 0.003,    // 位置依存も大
  "b_x": 0.8, "b_y": 0.9    // やや飽和的
}
```

## 旧モデルからの移行

### v1.0（線形モデル）からの互換性

旧パラメータ:
```json
{"shift_x_per_nm": 0.001, "shift_y_per_nm": 0.003,
 "noise_std_x": 0.002, "noise_std_y": 0.005}
```

新パラメータへの変換:
```json
{
  "a": 0.001,              // = shift_x_per_nm
  "b": 1.0,                // 線形なので1
  "c": 0.003,              // = shift_y_per_nm
  "d": 1.0,                // 線形なので1
  "noise_base_x": 0.002,   // = noise_std_x
  "noise_prop_x": 0.0,     // トルク依存なし
  "noise_base_y": 0.005,   // = noise_std_y
  "noise_prop_y": 0.0      // トルク依存なし
}
```

これにより、`b=d=1`, `noise_prop=0`で完全に旧モデルと等価になります。
